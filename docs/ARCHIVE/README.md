# 📦 Antigravity Document Archive

본 디렉토리는 더 이상 활성화되지 않거나, 역사를 기록하기 위해 보존된 문서들의 저장소입니다.

## 📁 디렉토리 구조
- **[reports/](reports/)**: 과거의 Gap Analysis, 인시던트 리포트, 실험 결과 등.
- **[issues/](issues/)**: 완료된 이슈 상세 문서 및 레거시 이슈 파일.
- **[governance/](governance/)**: 개정되기 전의 구버전 거버넌스 정책 및 감사 자료.
- **[ideas/](ideas/)**: 실현되지 않았거나 우선순위에서 밀린 씨앗 단계의 아이디어들.

## 📜 아카이빙 원칙 (`@/manage-docs`)
1. **비활성**: 3개월 이상 업데이트가 없거나 기능이 완전히 대체된 경우.
2. **기록성**: 결과 보고서, RCA(Root Cause Analysis) 등 성과 측정을 위해 보존이 필요한 경우.
3. **SSoT 이관**: 루트 `BACKLOG.md` 또는 `README.md`에 최종 상태가 반영된 구체 작업 내용.
