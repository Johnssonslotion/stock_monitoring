# 장기 전략 명세서 (Long-term Strategy Spec)

**작성일**: 2026-01-04
**담당자**: Doc Specialist & Architect

## 1. 개요
뉴스, 소셜 미디어 등의 **텍스트 데이터**를 수집/분석하여 시장의 심리(Sentiment)와 트렌드를 파악하고, 이를 기반으로 **섹터(Sector) 단위의 리밸런싱** 전략을 수행한다.

## 2. 데이터 수집 (Data Ingestion)
### 2.1 대상 소스
-   **뉴스 (News)**: 경제 전문 뉴스 RSS (매일경제, 한국경제 등), 네이버 금융 뉴스.
-   **소셜 (Social)**: Telegram 주요 주식 채널, 트위터(X) 금융 인플루언서 (API 허용 시).
-   **공시 (Disclosure)**: 전자공시시스템(DART) 주요 공시 (선택 사항).

### 2.2 수집 방식
-   **주기**: 1시간 단위 Polling (실시간성보다는 비용 효율성 중시).
-   **저장**: 원본 텍스트는 저장하지 않고, 분석된 메타데이터(날짜, 소스, 감성 점수, 관련 섹터)만 경량 DB에 저장.

## 3. 분석 로직 (Analysis)
### 3.1 NLP 파이프라인
1.  **전처리**: 불용어 제거, 중요 키워드 추출.
2.  **임베딩/분류**:
    -   경량 LLM (Gemini API, OpenAI API 등) 활용.
    -   **프롬프트 전략**:
        > "이 기사는 [반도체, 2차전지, 바이오, ...] 중 어떤 섹터와 관련이 깊은가? 호재인가 악재인가? (Score: -1.0 ~ 1.0)"
3.  **Aggregation**: 섹터별 일일 편향성(Bias) 점수 산출.

## 4. 전략 실행 (Execution)
### 4.1 리밸런싱 규칙
-   **주기**: 주 1회 또는 월 1회 (Long-term).
-   **조건**: 특정 섹터의 누적 감성 점수가 임계값(Threshold)을 초과할 때 비중 확대.
-   **포트폴리오**: 모의 투자 계좌 연동 (Long Only).

## 5. 지표 (Metrics)
-   **Hit Ratio**: 뉴스 감성 분석 결과와 실제 주가 흐름의 상관관계.
-   **MDD**: 최대 낙폭 관리.
